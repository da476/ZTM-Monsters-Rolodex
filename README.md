# KittyFriends Rolodex

This project pulls an assortment of images from Robohash.org to display in a searchable catalog of kitties. From the ZTM Academy course "Complete Web Developer" RoboFriends project.

## Tech Stack
- VSCode
- HTML, CSS, JavaScript
- React, Vite, NPM

## How to use KittyFriends
1. Open link in new tab.
2. Search by name or email.
